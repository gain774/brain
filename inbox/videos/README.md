# 📥 動画文字起こしの投入口

YouTubeの「文字起こしを表示」からコピーしたテキストを `.txt` でここに置く(またはチャットに直接貼る)と、
次のセッションで video-analyzer が解析し `research/videos/` に構造化して保存します。

ファイルの先頭に分かる範囲でメタ情報を書いてください(なくても可):

```
タイトル: AIを使って爆速でアプリ開発を行うためのコーディングワークフロー
チャンネル: Shin Coding Tutorial
URL: https://www.youtube.com/watch?v=3RFQZsgZ3nA
---
(ここから文字起こし本文)
```
